# Arch Dotfiles
1. Install the must have apps
```bash
pacman -S stow zsh tmux neovim starship ripgrep zoxide eza fzf lazygit
```

2. Set shell to `zsh` 
```bash
chsh -s $(which zsh)
```


# Ubuntu Dotfiles
1. Install the must have apps
```bash
sudo apt install stow zsh tmux neovim starship ripgrep zoxide eza fzf lazygit
```

2. Set shell to `zsh` 
```bash
chsh -s $(which zsh)
```
